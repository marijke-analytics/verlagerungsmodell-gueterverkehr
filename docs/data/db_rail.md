# Deutsche Bahn – Open Data (Schiene)

## Quelle
https://data.deutschebahn.com/

## Typische Datensätze
- Zugpositionen
- Verspätungsdaten
- Infrastruktur (Bahnhöfe, Strecken)
- Betriebsstellenverzeichnis

## Wichtige Spalten (je nach Dataset)
- **station_id**
- **station_name**
- **lat**, **lon**
- **train_id**
- **delay**
- **event_time**
