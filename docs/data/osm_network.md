# OpenStreetMap / OpenRailwayMap – Netzwerkdaten

## Quelle
https://www.openstreetmap.org/

## Typische Elemente
- **Nodes** – Punkte (Koordinaten)
- **Ways** – Straßen, Schienen, Wege
- **Relations** – komplexe Strukturen (Routen, Linien)

## Wichtige Attribute
- **highway** – Straßentyp
- **railway** – Schienentyp
- **maxspeed**
- **lanes**
- **bridge**, **tunnel**
